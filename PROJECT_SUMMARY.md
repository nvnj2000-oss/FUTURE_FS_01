
```bash
# Install dependencies
npm install

# Development server
ng serve
# Visit: http://localhost:4200

# Build for production
ng build

# Production build with optimizations
ng build --configuration production --optimization=true

# Deploy to Netlify
# Drag and drop dist/ folder to Netlify

# Deploy to Vercel
vercel

# Deploy to Firebase
firebase deploy
```
